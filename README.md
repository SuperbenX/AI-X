## 📌 项目简介
AI X 是一个 **开源共创项目**，目标是构建 **100+ 职业岗位的 AI 使用手册**。  
每个岗位文档提供：  
- 关键工作流程拆解  
- 常用 AI 工具推荐（入门 / 进阶 / 高阶）  
- 高频 Prompt 模板（可直接复制使用）  
- 实用案例与最佳实践  
- 常见坑 & 应急方案  
- 延伸资源  

👉 让 AI 小白也能一站式上手，让专业人士也能找到提升效率的新方法。
