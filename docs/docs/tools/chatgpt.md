## 💡 简介
由 OpenAI 推出的对话式 AI，可用于文本创作、总结、翻译、代码辅助等任务。

## 👥 适用人群
自媒体人, 创业者, 教育者, 程序员

## ✅ 推荐用途
- 文案创作
- 总结分析
- 办公提效
- 辅助写作 / 翻译

## 🌟 亮点功能
- 多语言支持
- 操作简单
- 整合插件丰富
- 支持结构化输出
