## 💡 简介
微软出品的视频剪辑工具，操作简单，适合初学者和日常视频内容制作。

- 文案撰写
- 多媒体内容创作
- 自动生成模板
- 播放演示 / 搜索答疑

## 🌟 亮点功能
- 模板丰富
- 输出质量高
- 适配多平台
