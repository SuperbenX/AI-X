## 💡 简介
集转录、剪辑、配音于一体的多功能音视频内容创作平台。

## 👥 适用人群
播客主, 剪辑师, 内容创作者

## ✅ 推荐用途
- 视频剪辑 / 生成
- 数字人营销内容
- 品牌视觉内容创作

## 🌟 亮点功能
- 多语种支持
- 简单拖放式操作
- 模板丰富
- 支持导出多格式
