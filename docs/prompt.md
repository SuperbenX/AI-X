# 📌 Prompt 学习路径

AI 新手往往会卡在“我不知道怎么写 Prompt”的门槛上。  
为了让小白快速上手，我们设计了一个 **三层进阶体系** —— 从照抄就能用，到掌握技巧，再到打造自己的 AI 工作流。  

---

# 🐣 Prompt 入门版块（小白必看）

## 1. Prompt 为什么这么重要？
- **Prompt = 驾驶方向盘**：AI 是引擎，Prompt 是方向盘。写得好，车开到你想去的地方；写不好，就会跑偏。  
- **写 Prompt 的好坏 → 决定 AI 的答案质量**。  
  - 好的 Prompt：清晰、具体、有条件 → 得到可用结果。  
  - 差的 Prompt：模糊、随意、无背景 → 得到笼统废话。  

👉 小白学习目标：从“随便问” → 学会“清晰提问” → 最终“像高手一样控制 AI 输出”。  

---

## 2. Prompt 的黄金公式
**角色 + 任务 + 条件 + 输出格式**

例子：  
`你是一名营销文案专家（角色），请帮我写一篇新品发布的朋友圈文案（任务），要求不超过50字，语气要热情亲切（条件），请给我3个版本（输出格式）。`

---

## 3. 入门对比示例

### 示例 1：写故事  
- ❌ 差的 Prompt：  
  `写一个小猫的故事。`  
  👉 输出：几句很平淡的故事。  

- ✅ 好的 Prompt：  
  `你是一名儿童绘本作者，请写一个适合5岁小朋友阅读的猫咪冒险故事，要求温暖治愈，字数在300字以内，并分3段落。`  
  👉 输出：有情节、有结构、符合目标读者。  

---

### 示例 2：写文案  
- ❌ 差的 Prompt：  
  `帮我写一个广告。`  

- ✅ 好的 Prompt：  
  `请为夏日冷饮品牌写一条广告文案，要求：①30字以内，②语气轻松活泼，③突出“清凉解暑”的感觉。请给我5个不同的版本。`  

---

### 示例 3：学习总结  
- ❌ 差的 Prompt：  
  `帮我总结这篇文章。`  

- ✅ 好的 Prompt：  
  `请帮我把以下文章总结成3个要点，每个要点不超过15字，方便我快速记忆。最后再给我一个50字的整体总结。内容是：……`

## 4. Prompt 入门技巧（小白必备）
1. **告诉 AI 它是谁** → “你是律师/设计师/老师”。  
2. **说清楚目标** → “写一封面试感谢信”。  
3. **设置条件** → “限制字数/风格/语气”。  
4. **要求格式** → “请用表格形式”。  
5. **要多个版本** → “给我3个不同思路的答案”。  

---

## 5. 小白常犯的错误 🚫
- **问太模糊** → AI 输出模糊。  
- **不给背景** → 结果不相关。  
- **不复盘** → 没有逐步改进 Prompt。  

👉 记住：Prompt 是可以“迭代”的。第一次结果不好 → 加条件 → 再试一次。  

---

## 6. 小白练习任务 📝
1. 写一段“自我介绍”，并分别改写成：正式版 / 幽默版 / 文艺版。  
2. 总结一篇新闻，用表格展示“时间 / 地点 / 事件 / 结果”。  
3. 为“咖啡店开业”写 3 条朋友圈文案（每条不超过40字）。  
4. 把一句话翻译成英文，并要求“商务语气”。  

---

## 7. 实用 Prompt 示例合集（初学者必备）

1. **写总结**：  
   `请帮我把以下内容总结成3个关键点：……`  

2. **写提纲**：  
   `请根据以下主题写一个大纲，每一节不少于3个小点：……`  

3. **改写语气**：  
   `请把这段文字改成幽默风格/学术风格/广告风格：……`  

4. **生成表格**：  
   `请把以下数据整理成表格，包含列：名称 / 时间 / 要点：……`  

5. **写邮件**：  
   `你是一名商务助理，请帮我写一封英文邮件，语气正式，内容是：……`  

6. **头脑风暴**：  
   `请给我10个适合夏天饮料品牌的创意 slogan。`

## 🟡 进阶玩法（掌握技巧）
> 目标：让用户理解如何更好地和 AI 对话。  
> 特点：技巧性、举例教学。

### 1. 常用技巧
- **角色扮演**：让 AI 扮演专家  
- **分步推理**：让 AI 先分析，再输出结果  
- **输出格式**：要求 AI 用 Markdown / 表格 / 代码块输出  
- **约束条件**：加上“不超过200字”“给出3种方案”等  

---

### 2. 进阶 Prompt 模板
- **先分析再总结**  
  ```
  请帮我分析[问题]，列出优缺点，最后给一个建议。
  ```

- **多角色协作**  
  ```
  你同时扮演[角色1]和[角色2]，先分别给出观点，再整合成一份综合报告。
  ```

👉 通过这些技巧，用户能逐渐写出更精准的 Prompt。

## 🔴 高手玩法（效率与创造力）
> 目标：让用户看到“高手怎么玩”，学会打造自己的 Prompt 库。  
> 特点：系统化、自动化、实战性。

### 1. 链式 Prompt（工作流）
示例：写文章  
1. 先生成提纲  
2. 展开每段内容  
3. 优化语言风格  
4. 排版成 Markdown  

---

### 2. 多模态 Prompt
- 结合 **ChatGPT + MidJourney + Figma 插件**  
- 示例：用 MJ 生成 moodboard → ChatGPT 写文案 → Figma 插件排版

---

### 3. 个性化 Prompt 调优
- 迭代方式：在原 Prompt 后加上“换个风格”“再详细一点”  
- 建立 **个人常用 Prompt 库**，形成属于自己的 AI 工作流
