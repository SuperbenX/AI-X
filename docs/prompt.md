## 1. 什么是 AI？🤖

- **一句话解释**：AI 就像一位超能助手，可以帮你写文章、画画、查资料、规划日程，甚至陪你聊天。
- **能做什么？**  
  ✅ 写作与办公：邮件、周报、PPT、大纲  
  ✅ 学习与研究：总结资料、出练习题、翻译  
  ✅ 创意与娱乐：绘图、短视频脚本、写小说  
  ✅ 生活助手：健身计划、饮食建议、旅行规划  

> 📝 记住：AI 不会取代你，而是帮你提升 10 倍效率。

## 2. 第一步体验 🌱

试试这些 **上手案例**，一分钟感受 AI 的威力：

1. **邮件助手**  
   ```
   请帮我写一封简短的请假邮件，理由是身体不适，收件人是我的上级。
   ```

2. **绘图助手**（MidJourney / SD）  
   ```
   A cute puppy sitting on the grass, watercolor style, bright colors.
   ```

3. **新闻总结**  
   ```
   请帮我总结今天的科技新闻，用 3 点简短说明。
   ```

## 3. Prompt 基础教学 ✍️

- **Prompt 是什么？**  
  就是你给 AI 的指令。像“魔法咒语”一样，问得好，效果就好。这是最重要的一步。  

- **技巧 1：角色扮演**  
   ```
   你是一名资深律师，请帮我解读下面的合同条款，重点标出风险点。
   ```

- **技巧 2：分步骤**  
   ```
   第一步，先帮我总结文章。  
   第二步，把总结翻译成英文。  
   第三步，生成一个推文格式。
   ```

- **技巧 3：输出格式**  
   ```
   请帮我输出一个表格，列出一周的健身计划，包含 日期 / 训练内容 / 饮食建议。
   ```

## 5. 场景化案例 🎯

- **学习场景**  
   ```
   请根据以下考试范围，帮我提炼 10 个重点知识点，并给出对应的练习题。
   ```

- **工作场景**  
   ```
   请帮我写一份 IT 部门的周报，包含本周完成的工作、下周计划和风险点，控制在 300 字内。
   ```

- **生活场景**  
   ```
   我想开始健身，每周 3 次训练，目标是减脂 + 保持肌肉。请帮我规划训练和饮食表。
   ```

- **创作场景**  
   ```
   请帮我写一段适合短视频口播的文案，主题是“如何高效学习英语”，控制在 60 秒内。
   ```

## 6. 常见问题解答（FAQ）❓

- **AI 会取代我吗？**  
  不会。它更像“外骨骼”，提升你的效率。用 AI 的人会取代不用 AI 的人。  

- **需要会写代码吗？**  
  不需要！大部分 AI 工具都支持自然语言输入。  

- **用 AI 会不会泄露隐私？**  
  要注意：不要输入敏感信息（身份证号、银行卡等）。  

- **免费和付费有什么区别？**  
  - 免费：速度慢、功能有限  
  - 付费：模型更强大、更快、更稳定

## 🟡 进阶玩法（掌握技巧）
> 目标：让用户理解如何更好地和 AI 对话。  
> 特点：技巧性、举例教学。

### 1. 常用技巧
- **角色扮演**：让 AI 扮演专家  
- **分步推理**：让 AI 先分析，再输出结果  
- **输出格式**：要求 AI 用 Markdown / 表格 / 代码块输出  
- **约束条件**：加上“不超过200字”“给出3种方案”等  

---

### 2. 进阶 Prompt 模板
- **先分析再总结**  
  ```
  请帮我分析[问题]，列出优缺点，最后给一个建议。
  ```

- **多角色协作**  
  ```
  你同时扮演[角色1]和[角色2]，先分别给出观点，再整合成一份综合报告。
  ```

👉 通过这些技巧，用户能逐渐写出更精准的 Prompt。

---

## 🔴 高手玩法（效率与创造力）
> 目标：让用户看到“高手怎么玩”，学会打造自己的 Prompt 库。  
> 特点：系统化、自动化、实战性。

### 1. 链式 Prompt（工作流）
示例：写文章  
1. 先生成提纲  
2. 展开每段内容  
3. 优化语言风格  
4. 排版成 Markdown  

---

### 2. 多模态 Prompt
- 结合 **ChatGPT + MidJourney + Figma 插件**  
- 示例：用 MJ 生成 moodboard → ChatGPT 写文案 → Figma 插件排版

---

### 3. 个性化 Prompt 调优
- 迭代方式：在原 Prompt 后加上“换个风格”“再详细一点”  
- 建立 **个人常用 Prompt 库**，形成属于自己的 AI 工作流
