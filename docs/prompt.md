# 📌 Prompt 学习路径

AI 新手往往会卡在“我不知道怎么写 Prompt”的门槛上。  
为了让小白快速上手，我们设计了一个 **三层进阶体系** —— 从照抄就能用，到掌握技巧，再到打造自己的 AI 工作流。  

---

# 🐣 Prompt 入门版块（小白必看）

## 1. Prompt 为什么这么重要？
- **Prompt = 驾驶方向盘**：AI 是引擎，Prompt 是方向盘。写得好，车开到你想去的地方；写不好，就会跑偏。  
- **写 Prompt 的好坏 → 决定 AI 的答案质量**。  
  - 好的 Prompt：清晰、具体、有条件 → 得到可用结果。  
  - 差的 Prompt：模糊、随意、无背景 → 得到笼统废话。  

👉 小白学习目标：从“随便问” → 学会“清晰提问” → 最终“像高手一样控制 AI 输出”。  

---

## 2. Prompt 的黄金公式
**角色 + 任务 + 条件 + 输出格式**

例子：  
`你是一名营销文案专家（角色），请帮我写一篇新品发布的朋友圈文案（任务），要求不超过50字，语气要热情亲切（条件），请给我3个版本（输出格式）。`

---

## 3. 入门对比示例

### 示例 1：写故事  
- ❌ 差的 Prompt：  
  `写一个小猫的故事。`  
  👉 输出：几句很平淡的故事。  

- ✅ 好的 Prompt：  
  `你是一名儿童绘本作者，请写一个适合5岁小朋友阅读的猫咪冒险故事，要求温暖治愈，字数在300字以内，并分3段落。`  
  👉 输出：有情节、有结构、符合目标读者。  

---

### 示例 2：写文案  
- ❌ 差的 Prompt：  
  `帮我写一个广告。`  

- ✅ 好的 Prompt：  
  `请为夏日冷饮品牌写一条广告文案，要求：①30字以内，②语气轻松活泼，③突出“清凉解暑”的感觉。请给我5个不同的版本。`  

---

### 示例 3：学习总结  
- ❌ 差的 Prompt：  
  `帮我总结这篇文章。`  

- ✅ 好的 Prompt：  
  `请帮我把以下文章总结成3个要点，每个要点不超过15字，方便我快速记忆。最后再给我一个50字的整体总结。内容是：……`  

---

## 4. Prompt 入门技巧（小白必备）
1. **告诉 AI 它是谁** → “你是律师/设计师/老师”。  
2. **说清楚目标** → “写一封面试感谢信”。  
3. **设置条件** → “限制字数/风格/语气”。  
4. **要求格式** → “请用表格形式”。  
5. **要多个版本** → “给我3个不同思路的答案”。  

---

## 5. 小白常犯的错误 🚫
- **问太模糊** → AI 输出模糊。  
- **不给背景** → 结果不相关。  
- **不复盘** → 没有逐步改进 Prompt。  

👉 记住：Prompt 是可以“迭代”的。第一次结果不好 → 加条件 → 再试一次。  

---

## 6. 小白练习任务 📝
1. 写一段“自我介绍”，并分别改写成：正式版 / 幽默版 / 文艺版。  
2. 总结一篇新闻，用表格展示“时间 / 地点 / 事件 / 结果”。  
3. 为“咖啡店开业”写 3 条朋友圈文案（每条不超过40字）。  
4. 把一句话翻译成英文，并要求“商务语气”。

# 🚀 Prompt 进阶玩法

当你已经掌握了基础 Prompt 技巧，现在是时候进入 **进阶玩法**。  
这一部分将教你如何系统化、工程化地使用 Prompt，解决更复杂的工作与创作任务。

---

## 1. 进阶心法
- Prompt 不只是“命令”，而是一种 **思维方式**。  
- 好的 Prompt 应该帮助 AI：  
  1. 明确上下文  
  2. 结构化任务  
  3. 允许迭代与优化  

---

## 2. 模块化 Prompt
将一个大任务，拆解为多个小 Prompt 模块，再逐步拼接。  

**示例**  
任务：写一篇公众号文章  
1. 模块 1：生成选题方向  
2. 模块 2：撰写提纲  
3. 模块 3：逐段生成内容  
4. 模块 4：整体润色与排版  

这种方式可以避免“一次性大而全”的失败，提升控制力。

---

## 3. 角色扮演进阶
给 AI 赋予多层身份，让它以专家的视角回答。  

**示例 Prompt**：
```
你是一位资深产品经理，同时也是一名心理学研究者。
请从这两个视角，分析用户为什么会沉迷短视频。
输出格式：
1. 产品经理视角
2. 心理学视角
3. 综合结论
```

---

## 4. 链式思维（Chain-of-Thought）
让 AI **一步一步思考**，而不是直接给结论。  

**示例 Prompt**：
```
请一步步分析以下逻辑题：
如果所有猫都会抓老鼠，而小白是一只猫，那么小白会不会抓老鼠？
在回答最终结论之前，请先逐条写下你的推理过程。
```

---

## 5. 多轮迭代 Prompt
学会用反馈优化结果，而不是一次性要求完美。  

**示例工作流**：
1. 第一次 Prompt：粗稿  
2. 第二次 Prompt：优化结构  
3. 第三次 Prompt：调整语气/风格  
4. 最终：输出成品  

---

## 6. 进阶 Prompt 示例集
- **市场分析**  
```
你是一位咨询顾问，请根据以下条件分析：  
目标行业：新能源车  
关注点：用户痛点、竞争格局、潜在机会  
请输出一份简明的 SWOT 分析表。
```

- **代码调试**  
```
你是 Python 高级工程师。  
请帮我检查以下代码中可能存在的逻辑错误，并逐行解释修改理由。
```

- **多语言翻译优化**  
```
请把以下中文翻译成英文，保留商务专业语气。  
然后再给出一个适合社交媒体传播的简短版本。
```

---

## 7. 练习任务
尝试以下练习，提升你的 Prompt 水平：
1. 选择一个工作任务（如写邮件、写方案） → 写 2 个不同 Prompt → 对比效果。  
2. 让 AI 扮演“两个不同角色”，让他们辩论同一问题 → 观察不同观点。  
3. 设计一个“模块化工作流”，比如写一份报告，从选题到排版分 5 步完成。

# 🎯 Prompt 高手玩法

## 1. 高阶理念：从“写 Prompt”到“设计系统”
- **小白 → 会写提示词**  
- **进阶 → 会优化任务、拆解模块**  
- **高手 → 设计 Prompt 系统 & 工作流**  
高手不再只是单次写 Prompt，而是：  
1. 设计一整套 Prompt 体系  
2. 把 Prompt 嵌入自动化工具 / 产品  
3. 结合 API、Agent、插件，形成生产力系统  

---

## 2. 高手必备技巧

### (1) Prompt 架构设计（Prompt Architecture）
- **思路**：像搭建软件架构一样，设计多层次、多模块的 Prompt。  
- **要点**：  
  - 上层：策略（目标、风格、约束）  
  - 中层：逻辑（步骤、思维链）  
  - 底层：执行（格式、输出模版）  
- **示例**：构建一份“市场调研自动报告 Prompt”，分为：  
  - 数据收集 → 分析模型 → 格式化输出 → 行业结论

---

### (2) Few-shot & Zero-shot 巧用
- **Zero-shot**：完全不给示例，AI 自主推理。  
- **Few-shot**：给 2-5 个示例，引导风格与逻辑。  
- **高手玩法**：  
  - 混合使用，既给风格样例，又留空白让 AI 自由发挥。  
  - 用「反例（bad examples）」让 AI 避开错误。  

---

### (3) 多代理协作（Multi-Agent Prompting）
- 把 AI 当作一个 **团队**，而不是单个助手：  
  - 一个 AI → 市场分析师  
  - 一个 AI → 财务专家  
  - 一个 AI → 品牌营销顾问  
- **玩法**：让多个 AI 在一个流程中对话 → 人类只做最终决策。  

---

### (4) 高级推理：自我反思（Self-Refine）
- **思路**：让 AI 对自己的答案进行批评和修改。  
- **Prompt 示例**：
  ```
  请你先完成任务A，然后写出对自己答案的批评，最后在批评基础上给出优化后的最终版本。
  ```
- **应用场景**：代码调试、学术写作、商业计划改进。  

---

### (5) 自动化与工具结合
- 把 Prompt **嵌入自动化流程**：  
  - Notion + AI → 自动生成日报  
  - Zapier/Make + AI → 自动生成内容并发布  
  - Python + OpenAI API → 自动批量跑 Prompt  

---

# 高手版块：AI + Prompt = 工作系统化

在完成了「小白入门」和「进阶玩法」的学习后，本章节将带你进入真正的高手领域。这里的核心理念是：**通过 Prompt 把工作流程自动化、模块化、体系化**。换句话说，不只是写好 Prompt，而是让 Prompt 成为你和 AI 协作的「工作引擎」。

---

## 1. 定位与思维转变

在高手阶段，你需要思考的问题是：
- 如何把一个完整的工作流程交给 AI 驱动？
- 如何让 AI 的输出形成**可复制、可扩展**的系统？
- 如何结合工具实现「AI + Prompt + 自动化」？

这里的关键不再是单条 Prompt，而是**Prompt 系统化思维**。

---

## 2. 核心结构

### A. Prompt 模块化思维
- **思路**：把一个大任务拆解为「输入 → 处理 → 输出」的小 Prompt 模块。
- **示例**：
  - **文章创作系统**
    1. 输入主题
    2. 输出多角度选题
    3. 输出文章大纲
    4. 输出逐段正文
    5. 输出风格优化稿
    6. 输出最终格式化文稿

这样，一个大任务就能被拆解为多个 Prompt 串联，形成一个标准化流程。

---

### B. Prompt + 工具整合
高手不仅写 Prompt，还要学会把 Prompt 嵌入到常用工具和平台中：
- **知识管理工具**：Notion / Obsidian → Prompt 模板库
- **自动化工具**：Zapier / Make.com → 定时触发 + Prompt 执行
- **团队协作工具**：Slack / 飞书机器人 → Prompt 驱动的智能助理

**示例**：
- 每天 9 点自动触发 → 调用 ChatGPT → 生成一篇行业动态 → 推送到 Slack 群组。

---

### C. Prompt 模板库搭建
高手会把 Prompt 沉淀为模板库，供个人和团队反复使用。
- **写作助手模板**：输入关键词 → 输出不同风格的文案。
- **设计助手模板**：输入主题 → 输出配色建议 + MidJourney 提示词。
- **电商助手模板**：输入商品 → 输出卖点提炼 + 五点描述 + 标题建议。

这样不仅自己高效，还能赋能整个团队。

---

### D. Prompt 驱动的工作流案例

#### 案例 1：自媒体人
- 选题 → 文章大纲 → 正文扩写 → 风格优化 → SEO 检查 → 自动上传到 Notion/博客。

#### 案例 2：电商运营
- 市场调研 → 商品卖点提炼 → 文案生成 → 详情页排版 → 自动推送至电商后台。

#### 案例 3：创业者
- 商业计划书草稿 → SWOT 分析 → 投资人问答模拟 → 路演稿优化。

---

## 3. 实战示例：自动生成文章并写入 Notion

```markdown
[系统提示：写作机器人]
目标：每天根据一个主题词，自动生成一篇文章并上传到 Notion。

Prompt 模块：
1. 主题解析 Prompt → 输入主题，输出 5 个不同角度
2. 大纲生成 Prompt → 基于选定角度，生成文章大纲
3. 正文生成 Prompt → 基于大纲，逐段生成内容
4. 风格优化 Prompt → 转换成目标风格（如幽默/专业/简洁）
5. 格式化 Prompt → 自动加标题、分段、引用

工具串联：
- Make.com：每天 9 点触发 → 调用 ChatGPT API → 写入 Notion
- Notion：作为文章库
```

---

## 4. 高手价值总结

- **小白学的是**：怎么问 AI。  
- **进阶学的是**：怎么和 AI 深度互动。  
- **高手要学的是**：怎么让 AI 替我干活，甚至帮团队干活。  

最终目标：把「Prompt」变成「工作系统」，实现真正的效率飞跃。

# 📚 精华版 Prompt 示例库 

以下示例严格遵循 **角色 + 任务 + 条件 + 输出格式** 的黄金法则，覆盖 13 大类场景，帮助小白到高手逐步掌握 Prompt 使用。

---

## 🏢 工作效率类

### 1. 邮件润色助手
- **Prompt**:  
  你是一个专业的商务邮件写作顾问。任务是帮我润色一封英文商务邮件，使其简洁、礼貌且具有专业感。条件：保持邮件长度不超过200词，避免复杂词汇。输出格式：提供修改后的邮件正文，并在最后用要点总结修改亮点。

### 2. 周报生成器
- **Prompt**:  
  你是一名企业管理顾问。任务是根据我提供的工作内容，生成一份简洁的周报。条件：字数控制在400字以内，使用条目化结构。输出格式：周报正文 + “下周计划”部分。

### 3. 会议纪要提炼
- **Prompt**:  
  你是一个会议纪要助理。任务是将输入的会议对话文本总结成纪要。条件：用“三段式”：会议主题、主要结论、待办事项。输出格式：Markdown 格式，带有小标题和编号列表。

### 4. 日程优化助手
- **Prompt**:  
  你是一个时间管理教练。任务是帮我优化一天的日程安排。条件：工作时段 9:00–18:00，午休1小时，优先保证高效工作时间。输出格式：表格形式展示优化后日程。

---

## 🎨 创意类

### 5. 广告文案生成器
- **Prompt**:  
  你是一个顶级广告创意总监。任务是为一个新饮料品牌撰写广告文案。条件：目标受众为18-30岁的年轻人，风格活泼，字数不超过100字。输出格式：3个不同风格的广告文案。

### 6. 品牌故事构建器
- **Prompt**:  
  你是一位品牌营销专家。任务是为一个帽子品牌编写品牌故事。条件：结合“潮流、DIY、玩乐”元素，带有情感共鸣。输出格式：500字品牌故事。

### 7. 短视频脚本助手
- **Prompt**:  
  你是一名短视频导演。任务是根据产品特性，生成一个30秒短视频脚本。条件：包含分镜头描述，突出卖点。输出格式：表格形式：镜头编号 | 描述 | 旁白。

### 8. 社交媒体爆款标题
- **Prompt**:  
  你是一个新媒体运营专家。任务是生成适合小红书的爆款标题。条件：结合热词，标题不超过20字。输出格式：一次生成10个标题。

## 📖 学习类

### 9. 知识点总结
- **Prompt**:  
  你是一名资深讲师。任务是把“财务自由之路”第一章的知识点总结出来。条件：条理清晰，分点列出，控制在300字内。输出格式：Markdown 格式要点列表。

### 10. 难点讲解
- **Prompt**:  
  你是一个高中数学老师。任务是用通俗易懂的方式讲解“函数极值”的概念。条件：用生活类比+公式。输出格式：讲解正文 + 简单例题。

### 11. 考试模拟题
- **Prompt**:  
  你是一名出题老师。任务是针对“Python 基础语法”设计5道选择题。条件：每题提供4个选项，标注正确答案。输出格式：题目编号 + 选项 + 正确答案。

### 12. 快速学习指南
- **Prompt**:  
  你是一个学习规划顾问。任务是帮我制定一份30天学习商务英语的计划。条件：每天不超过1小时，注重实用口语。输出格式：表格形式：天数 | 学习内容 | 目标。

### 16. 时间复盘助手
