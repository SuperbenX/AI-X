# 💰 Finance / Accounting（财务/会计）AI 实战指南

> 本指南面向 **财务/会计/出纳/FP&A/CFO 助理** 等岗位，帮助你用 AI 将「票据 → 记账 → 报表 → 预算与预测 → 税务合规」打通成一条高效链路。我们提供流程卡、工具表、实战 Prompt、常见坑与应急方案，**从 0 到 1 跑通你的数字化财务**。  
> 关键词：**财务自动化、OCR 票据识别、费用报销、对账、财报自动化、预算预测、FP&A、税务合规、AI 工具、Prompt 模板**。

## 🧭 核心流程（卡片式）

### 🧾 Step 1 — 票据/发票处理（OCR → 结构化）
- **工具**：Rossum / Mindee / ABBYY / 百度OCR / 腾讯云OCR  
- **操作要点**：  
  - OCR 提取「抬头、税号、金额、税额、供应商、日期」  
  - 对接科目/税码映射表，自动生成「入账建议」  
- **AI 加持**：异常字段高亮、凭证摘要自动生成、发票真伪校验提醒

---

### 💳 Step 2 — 费用报销与出纳记账
- **工具**：Expensify / Concur / 金蝶云星空 / 用友U8 / 飞书费用  
- **操作要点**：  
  - 报销单自动校验「预算可用额度 / 合规条款 / 发票重复」  
  - 自动生成付款批次与凭证草稿  
- **AI 加持**：报销合规模型（超标、重复、异常频次）→ 风险拦截

---

### 🧮 Step 3 — 账务处理与月结对账（GL）
- **工具**：Xero / QuickBooks / Sage Intacct / 金蝶KIS / 用友T+  
- **操作要点**：  
  - 银企对账自动匹配；长款/短款差异定位  
  - 固定资产折旧、预提/摊销自动分录建议  
- **AI 加持**：差异溯源（供应商、项目、成本中心）+ 关键信息摘要

---

### 📊 Step 4 — 管理报表自动化（P&L / BS / CF）
- **工具**：Power BI / Tableau / Looker Studio / FineBI  
- **操作要点**：  
  - 标准科目映射 → 模型口径统一（IFRS/US GAAP/企业会计准则）  
  - 报表模板化（多维：部门/项目/区域）+ Drilldown  
- **AI 加持**：报表文字洞察、差异解释（vs 预算/同期）

---

### 📈 Step 5 — 预算与预测（FP&A）
- **工具**：Anaplan / Pigment / Datarails / Cube / Excel+Copilot  
- **操作要点**：  
  - 驱动因子建模（人效、转化率、客单价、毛利、回款周期）  
  - 场景模拟（Base/Optimistic/Pessimistic）  
- **AI 加持**：自动生成「预算假设说明 & 经营逻辑图」

---

### 🧾 Step 6 — 税务合规与审计准备
- **工具**：Avalara / Stripe Tax / 本地电子税局 / 合规清单  
- **操作要点**：  
  - 税码校验、应纳税额估算、申报期提醒  
  - 审计底稿索引、凭证抽样、流程留痕  
- **AI 加持**：疑似税务风险提示（发票抬头/税号不一致、异常频次供应商）

## 💡 高频 Prompt 模板（卡片式，可直接复制）

### 📥 票据 → 凭证入账建议
```text
角色：会计助理  
任务：根据 OCR 提取的发票信息生成入账建议  
条件：字段包含 {抬头/税号/金额/税额/供应商/日期/发票类型}，参考科目映射表（简要列出）  
输出：表格 → 科目/税码/借贷方向/金额/摘要
```
**输入提示**：粘贴 OCR 结构化结果 + 科目映射片段。

---

### 🔁 银行流水自动对账
```text
角色：财务对账专员  
任务：对「银行流水」与「应收/应付明细」进行匹配  
条件：忽略小额差异 {阈值}；匹配优先级 {金额=日期>摘要关键字}  
输出：表格 → 流水编号/匹配单据/差异/处理建议
```
**输入提示**：贴上两份 CSV/表格的关键列样本。

---

### 📊 管理报表差异解释（vs 预算/同期）
```text
角色：FP&A 分析师  
任务：撰写本期 P&L 差异分析  
条件：输入实际值、预算值、同期值；维度含 {部门/项目/区域}  
输出：结构化解释（收入/成本/费用/毛利/净利），每项给「驱动因素 + 改进建议」
```

---

### 📈 现金流预测（13 周滚动）
```text
角色：资金经理  
任务：生成 13 周滚动现金流预测  
条件：输入应收账期、回款率、应付账期、采购节奏、税费与工资节奏  
输出：表格 → 周次/期初/现金流入/现金流出/期末 + 风险提示
```

---

### 🧠 驱动因子预算（Scenario Planning）
```text
角色：FP&A 规划顾问  
任务：基于关键驱动因子建立预算场景（Base/Optimistic/Pessimistic）  
条件：关键因子 {流量/转化率/客单价/毛利率/费用率} 的上下限  
输出：三表对比 + 关键假设说明 + 敏感性分析要点
```

---

### 🧾 税务申报检查清单
```text
角色：税务专员  
任务：生成当期税务申报检查清单  
条件：公司类型 {一般纳税人/小规模}；本期有/无跨境业务；是否有优惠政策适用  
输出：清单 → 申报表/附表/勾选对比/发票异常/滞纳风险
```

---

### 📝 CFO 月度经营备忘
```text
角色：CFO 助理  
任务：生成面向 CEO/董事会的月度经营简报  
条件：输入关键报表摘要 + 重点事项 + 现金流情况  
输出：三部分：核心指标摘要、风险与机会、下月行动建议（要点式 500–800 字）
```

## 🔎 实用案例（可复用）

### 案例 1：票据自动入账 + 月结提速  
- **场景**：票据量大、手工入账慢  
- **组合**：Rossum + Expensify + QuickBooks（或 金蝶/用友）  
- **打法**：OCR → 字段校验 → 科目映射 → 凭证草稿 → 批量过账  
- **成效**：入账效率 +60%，月结由 10 天缩短至 5 天

---

### 案例 2：13 周现金流预测 + 融资窗口判断  
- **组合**：Excel+Copilot / Datarails + Power BI  
- **打法**：回款/付款参数化 → 场景模拟 → 可视化看板  
- **成效**：提前 4–6 周识别现金缺口，调整融资/收款策略

---

### 案例 3：管理报表自动化 + 经营洞察  
- **组合**：总账系统 + Power BI + ChatGPT 洞察  
- **打法**：统一口径 → 多维报表 → AI 生成差异解释与行动建议  
- **成效**：周度经营复盘从 1 天降至 2 小时

---

### 案例 4：合规与审计准备  
- **组合**：Avalara/Stripe Tax + 审计底稿索引 + RPA  
- **打法**：税码/发票校验 → 底稿留痕 → 抽样与追溯  
- **成效**：应对审计时间缩短 30%，风险点更早暴露

## 🚀 进阶玩法（系统化升级）
- **RPA + OCR + GL 全链闭环**：UiPath/Power Automate 驱动凭证与报销自动流转  
- **驱动因子模型（Driver-based Model）**：以业务 KPI 驱动预算与预测  
- **异常检测**：基于历史分布/规则库识别「异常费用、异常开票、异常折扣」  
- **经营仪表盘（CFO View）**：一屏看「现金、毛利、费用率、周转效率、应收账龄」  
- **合规清单自动化**：不同税制/口径的申报清单模板化

## ⚠️ 常见坑 & 应急方案

### 坑 1：OCR 字段准确率不稳  
- **原因**：票据版式复杂/拍照质量差  
- **应急**：提高图片质量 → 打开「低置信度高亮」→ 建立字段校验规则（税号/金额比对）

### 坑 2：对账差异久拖未决  
- **原因**：匹配规则单一、跨系统口径不一致  
- **应急**：配置多条件匹配 & 冲销逻辑；设定差异阈值与升级规则

### 坑 3：报表口径混乱  
- **原因**：多套账、多币种、多部门缺统一映射  
- **应急**：制定「科目映射表 + 指标口径字典」→ 所有报表统一引用

### 坑 4：预算模型过于拍脑袋  
- **原因**：缺驱动因子与数据回测  
- **应急**：以「人效/转化/客单/毛利/周转」为驱动建立模型 → 回测校准

### 坑 5：税务/隐私合规风险  
- **原因**：上传敏感数据到外部模型  
- **应急**：优先企业版/私有化模型；敏感字段脱敏；保留处理留痕
