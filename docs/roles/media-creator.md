## ✍ 高频 Prompt 模板

### 1. 视频脚本生成
**Prompt**：
```
我需要为[平台]制作一个[类型]视频，目标观众是[人群]，视频时长[时长]，请帮我生成详细脚本。
```
**示例输入**：
> 我需要为抖音制作一个励志短视频，目标观众是大学生，时长 30 秒。  
**示例输出**：  
> 开头 5 秒：引人共鸣的问题；中段 15 秒：分享 3 个实用技巧；结尾 10 秒：号召点赞关注。

---

### 2. 标题优化
**Prompt**：
```
请帮我为以下内容生成 5 个高点击率标题，并按照吸引力排序。
内容：[内容概要]
```

---

### 3. 图片生成
**Prompt**：
```
请根据以下主题生成一张[风格]图片：
主题：[主题]
```

## 📂 实战案例

1. **短视频场景**：利用 ChatGPT 写脚本 → Canva 制作封面 → CapCut 剪辑 → 发布到抖音
2. **小红书图文**：用 ChatGPT 生成文案 → Midjourney 出图 → Canva 版面排版
3. **直播预告**：ChatGPT 编写直播话术 → Canva 制作宣传图 → 多平台同时发布

## 🗓 7 天入门计划

| 天数 | 任务 |
|------|------|
| 第 1 天 | 注册 ChatGPT、Canva、CapCut |
| 第 2 天 | 用 ChatGPT 生成 1 条脚本并尝试剪辑 |
| 第 3 天 | 设计 1 张视频封面图 |
| 第 4 天 | 用 Midjourney 生成 1 张主题图 |
| 第 5 天 | 尝试用 Make.com 或 Zapier 做简单自动化 |
| 第 6 天 | 产出并发布 1 条完整内容 |
| 第 7 天 | 回顾并优化工作流 |
