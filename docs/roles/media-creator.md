## 🧩 Prompt 1 · 短视频脚本生成（新手必用）  


#### 🧩 Prompt 2 · 爆款标题生成  
> 标签：#标题优化 #提升点击  ·  适用：短视频/图文/公众号  
**复制这个 Prompt：**  
```
围绕「{主题}」生成 10 个高点击标题（≤12 字），包含【数字】【对比】【悬念】三类各≥2条，并按吸引力排序。
```
**示例输入**：主题=AI 办公技巧  
**示例输出（节选）**：1）“3 天见效的 AI 提效术” 2）“别再瞎忙！这 3 招救你” …

---

#### 🧩 Prompt 3 · 图文排版与标题优化  
> 标签：#图文排版 #图文文案  ·  适用：小红书/公众号/知乎  
**复制这个 Prompt：**  
```
把下面内容改写为适合{平台}的图文：输出【≤16字标题】【3 段正文（短句）】【2 条配色/版式建议】。内容：{原文}
```
**示例输入**：平台=小红书；原文=（粘贴你的长段文字）  
**示例输出（片段）**：标题“7 天提效计划” → 分段正文 → 视觉建议：蓝白、16:9、手账风。

---

#### 🧩 Prompt 4 · 封面文案 + 视觉建议  
> 标签：#封面文案 #视觉创意  ·  适用：封面/海报缩略图  
**复制这个 Prompt：**  
```
为主题「{主题}」生成 5 条封面主标题（≤10 字）+ 1 条视觉建议（配色/元素/构图比例）。平台：{平台}；风格：{风格}。
```
**示例输入**：主题=AI 提升学习效率；平台=视频号；风格=商务简洁  
**示例输出（节选）**：标题“7 天提效计划”；视觉：16:9、蓝白、桌面+便签。

---

#### 🧩 Prompt 5 · 评论区互动问题  
> 标签：#评论互动 #提升停留  ·  适用：所有平台  
**复制这个 Prompt：**  
```
围绕「{主题}」生成 6 条评论区互动问题（≤15 字），语气{风格}，适配 {平台}。
```
**示例输入**：主题=时间管理；风格=轻松；平台=抖音  
**示例输出（节选）**：“你最浪费时间的习惯是？”“你更想省出早上还是晚上？”

---

#### 🧩 Prompt 6 · 跨平台改写（一稿多发）  
> 标签：#分发适配 #再利用  ·  适用：B 站/视频号/知乎/小红书  
**复制这个 Prompt：**  
```
请把下文改写成适合{目标平台}的风格，输出：标题 / 分行正文 / 结尾互动问题。保持信息不变但语气贴合平台。文本：{原文}
```
**示例输入**：目标平台=知乎；原文=（贴你的公众号内容）  
**示例输出（片段）**：更理性、更长段落、加引用与数据。

---

#### 🧩 Prompt 7 · 发布文案 + 标签  
> 标签：#发布描述 #话题标签  ·  适用：短视频/图文平台  
**复制这个 Prompt：**  
```
为{平台}生成 1 条发布文案（≤80 字）+ 10 个标签（长短词搭配），主题「{主题}」，受众{人群}，语气{风格}。请按检索量排序标签。
```
**示例输入**：平台=小红书；主题=提高拍照构图；人群=新手；风格=友好  
**示例输出（片段）**：文案一句话 + 标签列表（#构图技巧 #手机摄影 …）。

---

#### 🧩 Prompt 8 · 数据复盘与下一步计划（进阶）  
> 标签：#数据复盘 #内容优化  ·  适用：周更/日更复盘  
**复制这个 Prompt：**  
```
根据下列数据（标题/播放/完播/互动），总结：①表现最佳的 3 个主题②失败共性③下周 5 条实验建议（含预期指标）。数据：{粘贴数据}
```
**示例输入**：粘贴你的后台数据  
**示例输出（片段）**：主题 A 完播高 → 保留结构；主题 B 点击低 → 更换标题策略。

## 🗂️ 速查卡（Quick Reference）

> **适合：** 忙碌但想快速上手的初学者 / 临时查找灵感的创作者  
> **目标：** 用最少的时间找到最关键的指令、工具和避坑建议。

---

### 📌 核心要点
- **找工具**：先确定你是做视频、图文还是音频 → 对应去工具卡片查找。
- **用 Prompt**：按需求选择高频 Prompt → 直接复制到 AI 工具中试跑。
- **优化结果**：输出结果不满意 → 用“进阶 Prompt 卡片”里的改写模板再加工。

---

### ⚡ 立即行动
1. **选择一个目标**：例如“写一条视频文案”。  
2. **打开工具卡片** → 找到推荐的工具（如 ChatGPT + Canva）。  
3. **复制 Prompt** → 粘贴到 ChatGPT / DeepSeek，获取初稿。  
4. **快速修改** → 使用“进阶 Prompt”或手动润色，直接发布。  

---

### 🚫 避坑提醒
- **一次不要给 AI 太多指令** → 容易跑偏，分步给效果更好。
- **Prompt 太笼统** → 换成更具体的描述，AI 才能给精准输出。
- **忽略工具的更新** → 每月检查工具是否有新功能。
